# Setup und Entwicklungsumgebung

In dieser Lektion richtest du deine Entwicklungsumgebung ein und lernst die wichtigsten Tools kennen.

## Benötigte Tools

### 1. Code Editor

Wir empfehlen **Visual Studio Code**:

- Kostenlos und Open Source
- Viele hilfreiche Extensions
- Integriertes Terminal
- Git-Integration

**Download**: [code.visualstudio.com](https://code.visualstudio.com)

### 2. API-Zugang

Für praktische Übungen benötigst du Zugang zu AI-APIs:

**OpenAI**:
- Registriere dich auf [platform.openai.com](https://platform.openai.com)
- Erstelle einen API-Key
- Lade Guthaben auf (ca. 10€ reichen für den Kurs)

**Alternative**: Anthropic Claude API

### 3. Terminal/Kommandozeile

- **macOS**: Terminal (vorinstalliert)
- **Windows**: PowerShell oder WSL2
- **Linux**: Dein bevorzugtes Terminal

## Installation prüfen

Teste deine Installation:

```bash
# Node.js Version prüfen
node --version

# npm Version prüfen
npm --version

# Git Version prüfen
git --version
```

## Repository klonen

Lade das Kurs-Repository herunter:

```bash
git clone https://github.com/yourusername/ai-course-examples.git
cd ai-course-examples
npm install
```

## Umgebungsvariablen einrichten

Erstelle eine `.env.local` Datei:

```bash
# OpenAI API Key
OPENAI_API_KEY=sk-...

# Optional: Anthropic API Key
ANTHROPIC_API_KEY=sk-ant-...
```

⚠️ **Wichtig**: Teile niemals deine API-Keys öffentlich!

## Erste Tests

Teste deine Installation mit einem einfachen Script:

```javascript
// test.js
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

const completion = await openai.chat.completions.create({
  model: "gpt-4",
  messages: [{ role: "user", content: "Hallo AI!" }],
});

console.log(completion.choices[0].message.content);
```

Ausführen:
```bash
node test.js
```

## Troubleshooting

**Problem**: API-Key nicht gefunden
- **Lösung**: Überprüfe `.env.local` Datei und Pfad

**Problem**: Node.js Fehler
- **Lösung**: Stelle sicher, dass Node.js Version ≥ 18 installiert ist

**Problem**: Rate Limit Error
- **Lösung**: Warte kurz und versuche es erneut
